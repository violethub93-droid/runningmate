러닝메이트 — 클로바 음성 연결 실행 패키지 (코치형 + 친구형)
================================================================

[구성]
  index.html        ← 앱 (프로토타입 v3 + 클로바 음성 재생)
  coach/*.mp3        ← 코치형 클로바 음성 (55개)
  friend/*.mp3       ← 친구형 클로바 음성 (43개, 멘트풀은 추후 확장 예정)

[실행 방법 — 둘 중 하나]

1) 데스크톱에서 빠르게 확인 (음성 O / GPS X)
   - VS Code에서 이 폴더 열기 → index.html 우클릭 → "Open with Live Server"
   - 또는 index.html을 크롬에 드래그.
   - 시작 화면의 "GPS 없이 시뮬레이션" 으로 음성·엔진 확인.
   - 코치형 / 친구형을 바꿔가며 같은 러닝이 어떻게 다르게 들리는지 비교.

2) 폰에서 실제 러닝 (음성 O / GPS O) ★현장 테스트
   - GPS와 오디오는 HTTPS에서만 동작합니다. 아래 중 하나로 호스팅:
     · Netlify Drop (app.netlify.com/drop): 이 폴더를 드래그&드롭 → 즉시 https 주소
     · GitHub Pages: 저장소에 올리고 Pages 활성화
   - 폰 브라우저로 https 주소 접속 → 위치 권한 허용 → 러닝 시작.

[확인 포인트]
  - 상단 "오디오 ✓/✗" 카운터: ✓ 클로바 재생 / ✗ 파일없어 TTS 대체.
  - 코치형·친구형 모두 ✓만 올라가야 정상 (검증 완료: 누락 없음).
  - 발화 로그에 멘트마다 "클로바 / TTS" 표시.

[3차 테스트 권장 세팅]
  - 민감도: 보통~둔감 / 전역 발화 간격: 12·20·30 조합 바꿔보며 최적 구간 찾기.
  - 실제 계단·오르막 구간에서 원시 vs 스무딩 페이스 차이 관찰.

[친구형 멘트풀 확장 시]
  - 새 mp3를 friend/ 에 규칙대로 추가 + index.html의 SIT 친구형 배열에 한 줄 추가.
